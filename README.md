# 📚 Topics Covered
## 🔹 Class Work
These are the implementations, algorithms, and exercises coded during lecture sessions:

- **Linked List**: Singly, doubly, or circular linked list implementations  and operations.

- **Recursion**: Classic mathematical and logical problems solved using recursive functions.

- **Sorting and Searching**: Core algorithms like Bubble Sort, Merge Sort, Quick Sort, Binary Search, etc.

- **Stack**: LIFO (Last-In, First-Out) implementations using arrays or linked lists.

- **Tree**: Hierarchical structures including Binary Trees and Binary Search Trees (BST).

## 🏠 Home Work
Independent assignments and practice problems completed outside of class to reinforce theoritical concepts:

- **Linked List**: Linked List: Advanced node manipulation and edge-case challenges.
- **Queue**: FIFO (First-In, First-Out) operations, circular queues, or dequeues.

- **Recursion**: Complex problem-solving exercises.

- **Stack**: Bracket matching, infix-to-postfix conversions, and stack-based utility problems.
